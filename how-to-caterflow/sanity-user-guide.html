How to Use Caterflow by Synapse

A simple guide to managing your inventory with Sanity

Getting Started

Welcome to Caterflow by Synapse! This system helps you manage inventory across multiple sites, track orders, and streamline your operations.

1
Log in to the system

Use the option to log in with Google account. You can only access the dashboard with a pre-approved Google email address.

2
Familiarize yourself with the sidebar

The left sidebar is organized into three main sections: Master Data, Inventory Transactions, and System Administration.

3
Start with Master Data

Before adding transactions, set up your core data: Sites, Storage Bins, Categories, Stock Items, and Suppliers.

Setting Up Your Master Data

Master data forms the foundation of your inventory system. Follow these steps in order:

1
Add Your Sites

Sites are physical locations (e.g., "Main Kitchen", "Cafe", "Warehouse").

  • Go to Master Data → Sites
  • Click "Create new"
  • Add a name, code, and location description
  • Save your site
2
Create Storage Bins

Bins are specific storage locations within sites (e.g., "Shelf A3", "Cold Room 1").

  • Go to Master Data → Storage Bins
  • Click "Create new"
  • Name your bin and select which site it belongs to
  • Specify the bin type (shelf, refrigerator, etc.)
  • Add a location description for easy finding
3
Set Up Categories

Categories help organize your stock items (e.g., "Dry Goods", "Perishables", "Cleaning Supplies").

  • Go to Master Data → Categories
  • Click "Create new"
  • Add a title and description for your category
4
Add Suppliers

Suppliers are companies you purchase from.

  • Go to Master Data → Suppliers
  • Click "Create new"
  • Add the supplier name, contact person, and details
5
Create Stock Items

Stock items are the products you keep in inventory.

  • Go to Master Data → Stock Items
  • Click "Create new"
  • Add item name, SKU, category, and unit of measure
  • Set minimum stock levels for alerts
  • Assign a preferred supplier
Tip: You can view all stock items or filter them by category using the sidebar options.

Managing Inventory Transactions

Once your master data is set up, you can start managing inventory movements.

Purchase Orders

Create orders to send to suppliers when you need to restock items.

  • Select the supplier
  • Add items and quantities
  • Track status from ordered to received

Goods Receipts

Record items when they arrive from suppliers.

  • Link to purchase orders
  • Specify receiving bin location
  • Note item condition and expiry dates

Dispatch Logs

Track items sent from one site to another.

  • Select source bin
  • Choose destination site
  • Record dispatched quantities

Internal Transfers

Move items between bins within the same site.

  • Select from and to bins
  • Record transferred quantities
  • Update inventory locations

Stock Adjustments

Correct inventory counts for losses, damages, or found items.

  • Select adjustment type (loss, wastage, etc.)
  • Record quantities and reasons
  • Maintain accurate inventory records

Inventory Counts

Perform physical counts and compare with system records.

  • Select bin to count
  • Record counted quantities
  • System shows variances automatically

System Administration

1
Manage Users

Add team members and assign appropriate roles.

  • Go to System Administration → App Users
  • Click "Create new"
  • Add user details and assign role
  • Site Managers need an associated site
2
Configure Notifications

Set up alerts for low stock, expiring items, etc.

  • Go to System Administration → Notification Preferences
  • Enable/disable notification types
  • Set threshold values
  • Choose delivery methods (email, SMS, in-app)

Best Practices

Keep Data Consistent

Use consistent naming conventions for sites, bins, and items to make searching easier.

Regular Inventory Counts

Schedule regular counts to keep your digital records accurate.

Train Your Team

Ensure all users understand how to record transactions properly.

Use the Search Function

The search bar at the top helps you quickly find items, orders, or transactions.

Important: Always record transactions as they happen to maintain accurate inventory levels.
