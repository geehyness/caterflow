# Caterflow User Guide

I'll create a comprehensive, multi-page user guide for your Caterflow inventory management system. The guide will be interactive, user-friendly, and highlight important features without being overly technical.

## User Guide Structure

Let me create a complete user guide with the following pages:

1. **Getting Started** - Introduction and basic navigation
2. **Master Data Management** - Sites, bins, categories, suppliers, and stock items
3. **Inventory Operations** - Purchases, receipts, dispatches, transfers, adjustments
4. **Reporting & Analytics** - Dashboard, stock levels, activity tracking
5. **System Administration** - User management, notifications, settings

        // Simple navigation highlighting

            link.addEventListener('click', function(e) {
                document.querySelectorAll('.nav-item a').forEach(l => l.classList.remove('active'));
                this.classList.add('active');
                
                // Scroll to section
                const targetId = this.getAttribute('href');
                const targetSection = document.querySelector(targetId);
                if (targetSection) {
                    e.preventDefault();
                    window.scrollTo({
                        top: targetSection.offsetTop - 100,
                        behavior: 'smooth'
                    });
                    
                    // Update URL hash
                    history.pushState(null, null, targetId);
                }
            });
        });
